### 12 Design Ideas for Map & Catalog
1. **Filter sidebar** — Tags, difficulty, themes, type checkboxes
2. **Add-to-map toggle** — Each attraction has a checkbox; map stays clean by default
3. **Color-coded by category** — Each type gets a distinct color
4. **Difficulty badges** — Easy/Moderate/Strenuous/Drive-up
5. **In-between vs near-stop** — Two distinct tag types
6. **Cost function display** — Show actual driving detour cost per stop
7. **Sources page** — APA citations with footnote links on cards
8. **Theme-aware UI** — Catalog colors change with active theme
9. **Search bar** — Filter by name or keyword
10. **Sort options** — By detour time, cost, rating, distance
11. **Compact/list view** — Toggle between card grid and list
12. **Map techniques** — Clean markers, clustering, fit-bounds, layer switch
