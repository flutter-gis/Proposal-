# 💍 J & Dee — The Wilderness Romance

A 3D interactive PWA for a 6-day New Hampshire wilderness road trip & proposal.

## ✨ Features
- 3D engagement reveal (Three.js) — ring box opens in a sunset forest
- 20-instrument Web Audio sound engine with 10 composed pieces
- 10 dynamic themes (time-of-day + love-themed) with full CSS variable system
- 12 dynamic PWA icons that change with the time of day
- 6-page app: Home, Trip, Map, Proposal, Us, Settings
- History API routing with back button support
- Offline-capable PWA with service worker
- Accessible (WCAG 2 AA, 0 violations)

## 🚀 Quick Start
```bash
npm install
npm run dev
```

## 🛠️ Tech Stack
- Next.js 16 + React 19 + TypeScript
- Three.js + React Three Fiber (3D reveal)
- Tailwind CSS v4 (themed design system)
- Web Audio API (procedural music synthesis)
